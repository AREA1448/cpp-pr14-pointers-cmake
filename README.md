# Практична робота №14 - Робота з вказівниками (C++ + CMake)

## Завдання
Реалізовано три функції з обов’язковим використанням вказівникової арифметики:
1. Обчислення середнього парних та непарних елементів.
2. Пошук найдовшого та найкоротшого рядка.
3. Обчислення суми елементів у зворотному порядку.

## Як зібрати та запустити
```bash
mkdir build && cd build
cmake ..
cmake --build .
./pointers_work14
